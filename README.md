Simple python web app

Overview
This is a simple web application build with python to practice core web development such as:
1)Routing
2)Handling requests
3)basic work with memory and db.
The goal of the project was to understand how web apps works and serve as a starting poiner.

Features
1) Basic routes to demonstrate http methonds(get,post,put,delete)
2) Simple data model to represent items.
3) Separation of code into multiple files

Installation
1> clone the repository:

  git clone https://github.com/NickStamatelatos/webSimpleProject.git
  cd webSimpleProject

2> Create and activate a virtual environment (this commands is for windows):
  python -m venv .venv
  .venv\Scripts\activate
3> Install dependencies
  pip install -r requirements.txt
